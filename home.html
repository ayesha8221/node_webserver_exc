Home page

'/' (home page): When a user visits the root path (/), the server serves the content of the 'home.html' file.

/about: When a user visits the /about path, the server serves the content of the about.html file.

/contact: When a user visits the /contact path, the server serves the content of the contact.html file.

/products: When a user visits the /products path, the server serves the content of the products.html file.

/product/:id (single product page): When a user visits a path that matches /product/:id, the server serves the content of the product.html file. Here :id is a placeholder for a specific product's identifier, and it will be used to fetch data for the corresponding product page. Note that this placeholder :id could be any valid product identifier, like a product number or a product name.
